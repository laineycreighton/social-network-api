## Description

Welcome to the Social Network API Backend project! This repository contains the code and documentation needed to build the backend infrastructure for a social networking platform. With this API, you can create, manage, and interact with user profiles, thoughts, and reactions.

<br>

## Learning Points

- How to use NoSQL.
- How to use MongoDB.
- How to use Mongoose.
- How to use Postman to test routes.
